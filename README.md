# Mobile App

This contains the code to build the mobile application in the project. It is deployed on Google Play Store at:

https://play.google.com/store/apps/details?id=com.fypproject.tradeshow_guidance_app

# Installation

The Flutter SDK is required to run this application. 

To install, go here:

https://docs.flutter.dev/get-started/install

Follow the instructions on the Flutter page that will tell you how to use Android on your respective operating system.


To run the application, Visual Studio Code is used.

To install, go here:

https://code.visualstudio.com/download


Once Flutter SDK is installed, run `flutter pub get` to install necessary packages from main directory.

# Launching the app

The application can be run from lib/main.dart in Visual Studio.

Or running `flutter run` from main directory.
